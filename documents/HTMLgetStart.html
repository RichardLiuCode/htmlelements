Get Start For HTML

What is HTML?

HTML or Hyper Text Markup Language is the code language with many tags that mostly used to display a website.

What does HTML use for developing?

HTML can use for developing:

  1. Website UI
  2. Browser Extension
  3. Desktop/Mobile Application

Example of HTML

<!DOCTYPE html>
<html>
  <head>
    <title>HTML website</title>
  </head>
  <body>
    <h1>Hello!</h1>
    <hr>
    <p>This is a simple website</p>
  </body>
</html>
