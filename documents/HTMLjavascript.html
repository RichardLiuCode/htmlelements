Run Javascript in HTML documnet

Run javascript in a HTML document can make the HTML document has more feature other than just display text or elements only, elements such as button, input may not work without javascript.

Run Javascript in HTML by using attribute

You can put these attribe in html to run javascript.

  1. onload

    Example: 

    <body onload="console.log('content loaded!')">button</body>

  2. onclick

    Example: 

    <button onclick="console.log('button clicked!')">button</button>

  3. onmouseover
  4. onmouseout

Run Javascript in HTML by using a script tag

You can write your javascript code in the script tag and you code will be executed when thr browser load the web page

<script>
  console.log("Code loaded!");
</script>

OR

Use the src attribe in the script tag

<script src="script.js"></script>

The code above will import the file with the filename "script.js" and execut it.
